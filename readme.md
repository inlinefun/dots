### The repository is structured with [stow](https://www.gnu.org/software/stow/). 
Each folder is a [stow](https://www.gnu.org/software/stow/) package. Installation is as simple as "stowing" the required package.

> Usage is as simple as running `stow <package>` where `package` can be any folder in the repo
